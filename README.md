# AI-Powered-Sliding-Puzzle-Taquin-with-A-Search-


An interactive 8-puzzle game built with Python and Tkinter, featuring an automated solver powered by the **A* Search Algorithm**.

## 🚀 Features
- **Interactive Gameplay:** Click on tiles adjacent to the empty space to move them manually.
- **AI Solver:** Watch the A* algorithm solve the puzzle in real-time.
- **Adjustable Speed:** Control how fast the AI makes moves using a slider.
- **Heuristic Search:** Uses the "Misplaced Tiles" heuristic to find the shortest path to the solution.

## 🛠️ Technologies Used
- **Python 3**
- **Tkinter** (for the Graphical User Interface)
- **A* Algorithm** (Artificial Intelligence)
